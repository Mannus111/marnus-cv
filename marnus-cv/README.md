# Marnus de Beer | Virtual CV & Portfolio

A modern, responsive, and interactive online portfolio/CV for **Marnus de Beer**, final-year BSc Information Technology student at North-West University.

![Portfolio Preview](/images/PortfolioScreenshot.png)

## ✨ Features

- **Clean & Modern Design** with smooth animations and particle background
- **Fully Responsive** – optimized for desktop, tablet, and mobile
- **Interactive Navigation** with mobile hamburger menu
- **Downloadable CV** (PDF)
- **Sections**:
  - Hero with quick contact
  - About Me
  - Education (with stats and coursework)
  - Projects
  - Skills (with progress bars)
  - Experience
  - Certifications
  - Contact
- **Accessibility** focused (ARIA labels, semantic HTML)
- **Performance** optimized with preloaded fonts and minimal dependencies

## 🛠️ Technologies Used

- **HTML5** – Semantic markup
- **CSS3** – Custom styles with variables, animations, and responsive design
- **JavaScript** – Interactive elements, stats counter, mobile menu, particles
- **Google Fonts** – Inter & JetBrains Mono
- **SVG Icons** – Inline for performance
